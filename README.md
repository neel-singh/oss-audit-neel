## 📌 Description
This repository contains a collection of Bash scripts developed as part of an Open Source Software assignment. These scripts demonstrate system auditing, package inspection, log analysis, and open-source principles.

---

## 📂 Scripts Included

### 🔹 Script 1: System Identity Report
Displays system details like kernel, user, uptime, and distribution.

### 🔹 Script 2: FOSS Package Inspector
Checks if a package is installed and describes it.

### 🔹 Script 3: Disk and Permission Auditor
Analyzes directory permissions and disk usage.

### 🔹 Script 4: Log File Analyzer
Searches logs for keywords and displays matches.

### 🔹 Script 5: Open Source Manifesto Generator
Generates a personalized open-source manifesto.
